# L08 — Palette consistency criteria template

## What this lesson claims
All shots within one world-time share one named palette; cross-world-time
palettes can differ but same world-time must not drift.

## Scenario requirements
- ≥2 shots belong to the same "world-time" (same location, same time
  of day, same world-context).
- Optional: ≥1 shot in a different world-time to test separation.

## Judge rubric

### Criterion 1: Same-world-time palette consistency
All shots in the same world-time group share a recognizable palette.
Judge describes the palette in 1-2 adjectives ("warm amber" / "cool
teal") and verifies each shot fits it.

Pass: all same-world-time shots match.
Fail: any shot's palette reads as a different world-time.

### Criterion 2 (if multi-world-time): Cross-world-time differentiation
Shots in different world-times have palpably different palettes (not
just slight variations).

### Overall
Pass if Criterion 1 passes. (Criterion 2 is optional if only one
world-time.)

- Specify world-time groups and the target palette per group.
